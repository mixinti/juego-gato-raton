# juego-gato-raton
¿Qué creé?:
Un simulador del juego Gato y Ratón usando el algoritmo Minimax con profundidad limitada.

¿Cómo funciona?:
Tablero 5x5.
Gato minimiza distancia.
Ratón maximiza distancia.
Se usa distancia Manhattan como función de evaluación.

Qué funcionó:
El gato aprende a acorralar.
El ratón huye inteligentemente.

Qué fue difícil:
Entender la recursividad del Minimax.
Evitar que el algoritmo sea demasiado lento.

Momento “ajá”:
Cuando mi código por fin funcionó.
